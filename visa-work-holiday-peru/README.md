# VisaWork Peru 🇵🇪🇦🇺

Plataforma interactiva completa para ciudadanos peruanos que quieren aplicar a la **visa Work and Holiday Australia 2026**.

## 🚀 Funcionalidades

- **Checklist interactivo** - Marca documentos completados con guardado local
- **Calculadora de costos** - Estimacion total en soles y dolares
- **Recomendaciones personalizadas** - Basadas en tu progreso
- **Guia completa IELTS** - Estructura, vocabulario, gramatica y recursos
- **Simulador de examen** - Practica con las 4 secciones y cronometro
- **Diseño responsivo** - Funciona en celular, tablet y desktop

## 🌐 Como Subir a GitHub Pages

1. Crear repositorio en GitHub
2. Subir esta carpeta `visa-work-holiday-peru/`
3. Ir a Settings > Pages
4. Seleccionar rama `main` y carpeta `/visa-work-holiday-peru`
5. Tu pagina estara en: `https://tu-usuario.github.io/visa-work-holiday-peru/`
